# DECISIONS-B — оркестраторские решения по конфликтам Фазы A

Статус: ПРИНЯТО. Фикс-агенты Фазы B обязаны применить эти решения. При противоречии этого файла и любой спеки — побеждает этот файл; канон обновляется соответственно.

| # | Конфликт | Решение |
|---|---|---|
| K1 | Нумерация файлов в перекрёстных ссылках не совпадает с фактами | Фактические имена файлов на диске = истина. Файлы НЕ переименовывать. Все внутренние ссылки во всех доках привести к фактическим именам. В 00-canon добавить раздел «File Map» с каноническим списком файлов. |
| K7 | Финал ивента: «Вс 21:00» (дек) vs «Вс 20:00 UTC» (канон) | Фиксируем: **Вс 20:00 UTC** круглогодично (=23:00 МСК, 16:00 ET, 13:00 PT). Все доки приводятся. |
| K2 | Pity Prize Machine: единый 30 (economy) vs двухступенчатый (monetization) | Побеждает 15-monetization: **Rare гарант ≤10, Chase гарант ≤40**, дроп 68/24/6.5/1.5%. 14-economy приводится. |
| K5 | Порог мерджа <25 DAU vs стабильность ивента от ~40 | **Мягкий флаг с 7-дневного среднего DAU<40** (баннер+стимулы), **активное предложение-мердж с <25**. В 10-server-event добавить автоскейл вех ивента от фактического DAU города (если отсутствует). |
| K4 | Barn/Coop 5 уровней vs рамка построек 1–10 | Рамка 1–10 из 13-progression — мастер. 03-animals расширяет таблицы вместимости до уровней 1–10. |
| K3 | Smoker 480 мин vs T5-стандарт 300 | Канон §2.2 переформулировать как диапазон: T5 = 240–480 мин (стандарт 300). Smoker 480 остаётся — ночной цикл это фича. |
| K9 | «Два грузовика» в RU UI | Экспедиционный = EN **Route Truck**, RU «грузовик». Переезд = EN **Moving Van**, RU «фургон переезда». Канон-реестр и все доки привести. |
| K8 | Окно ярмарки 36 vs 48 ч | Подтверждаю **36 ч** (Сб 00:00 UTC → Вс 12:00 UTC). Плейтест-вопрос остаётся в OPEN-QUESTIONS. |
| 12-O1 | Ускорение кулдауна переезда за Dimes | **НЕТ.** Канон D11 уточнить: монетизация удобства НЕ распространяется на социальные кулдауны (переезды, голосования, менторство) — анти-абуз миграций перед ивентами. |
| K11 | know_how_points как 5-я валюта в currency_ledgers | Не валюта. В 20-backend хранить в состоянии know-how игрока (player_know_how/state-таблица), из currency_ledgers исключить. Канон: ровно 4 валюты. |
| K10 | Кривая цены земли ×1.18/слот vs «×2.2/уровень» | Мастер-формула = 02-farm: **×1.18 за слот**. Канон-формулировку заменить на «×1.18/слот (≈×2.2 на пакет ~5 слотов)». |
| 03-O1/O3 | Apiary/Beekeeper вне канона | **Apiary добавить** в канон-список построек (9-я). Beekeeper — НЕ отдельный архетип: обязанности пасеки у фермерского архетипа (Farmhand). Ростер остаётся 12. |
| 09/02/04-кэпы | Разные дневные кэпы бустеров в 4 доках | Единая таблица дневных кэпов бустеров живёт в **14-economy** (мастер). Остальные доки ссылаются, локальные цифры убрать. |
| 13-ОВ7/8 | Farm Value: баланс весов | Формула живёт только в 13-progression. Совокупный вклад косметики+коллекций **капится 15%** агрегата. 14-economy дублирующую формулу заменить ссылкой. |
| 11-O1 | Статус 11-town | 11-town.md — единый мастер-док соцсистем. Ссылки на несуществующие под-спеки заменить ссылками на разделы 11-town. |

Общие правила фиксов: файлы не переименовывать; git не трогать (коммитит оркестратор); правки — минимально-инвазивные, но конфликт должен быть закрыт полностью; после правки дока его раздел «Открытые вопросы» актуализировать (закрытые — удалить, с пометкой решения из этого файла).
